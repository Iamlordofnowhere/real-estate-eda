#  EDA: Анализ рынка недвижимости России 2021

## Описание
Исследовательский анализ данных рынка недвижимости России за 2021 год.  
Датасет содержит 11.3 млн объявлений о продаже квартир по всей России.

## Цель
Найти закономерности в ценах на жильё — по типу объекта, этажу, площади и динамике за год.

## Инструменты
- Python, pandas, numpy
- matplotlib, seaborn

## Ключевые выводы
- Цена за м² выросла на **+34%** за 2021 год (с 68к до 91к руб)
- Новостройки дороже вторички на **30%** (99к vs 76к руб/м²)
- Первый этаж дешевле средних этажей на **28%**
- Площадь квартиры почти не влияет на цену за м² (корреляция 0.11)

## Данные
[Russia Real Estate 2021 — Kaggle](https://www.kaggle.com/datasets/mrdaniilak/russia-real-estate-2021)
